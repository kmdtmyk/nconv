## Install

```
npm install @kmdtmyk/nconv
```

## Usage

```js
import Nconv from '@kmdtmyk/nconv'

Nconv.romajiToHiragana('nihongo')
// => にほんご

Nconv.romajiToZenKata('nihongo')
// => ニホンゴ

Nconv.romajiToHanKata('nihongo')
// => ﾆﾎﾝｺﾞ
```
